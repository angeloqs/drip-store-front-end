## 🚀 Como Executar o Projeto

### 💻 Pré-requisitos
- Node.js (v18+)
- npm (v9+) ou yarn

### 🔧 Execução
```bash
# Clone o repositório
git clone https://github.com/angeloqs/drip-store-front-end.git

# Instale as dependências
npm install

# Inicie o ambiente de desenvolvimento
npm run dev

# Acesse o endereço e visualize a página no navegador
http://localhost:5173
```
